# Direction

- Personality: precise, elegant, technical
- Density: balanced
- Color world: cool graphite neutrals with restrained electric blue emphasis
- Depth model: layered surfaces with crisp borders and subtle glow
- Signature elements: atmospheric grid background, editorial hero typography, quiet motion on reveal and hover

# Design Principles

- Prioritize credibility through hierarchy and restraint.
- Use accent color to guide attention, not decorate sections.
- Keep sections readable on desktop and mobile without changing the visual language.
- Prefer strong typography and spacing over heavy ornamentation.

# Color Tokens

- Background: near-black graphite
- Surface: dark steel panels with translucent overlays
- Border: low-contrast cool borders
- Primary text: high-contrast white
- Secondary text: muted slate
- Accent: electric blue for focus points and CTAs

# Type Scale

- Display: bold, tight tracking, used only in hero
- Heading: semibold to bold with strong contrast
- Body: medium contrast, comfortable line length
- Eyebrow and labels: uppercase with increased letter spacing

# Spacing Scale

- Base rhythm uses 8px increments
- Section padding expands substantially on desktop and compresses cleanly on mobile

# Radius Scale

- Large panels: 28px
- Cards: 22px
- Pills and buttons: 999px or 16px depending on role

# Border Strategy

- Panels rely on soft internal highlights and restrained outer borders
- Accent glow is reserved for hero highlights and primary CTA

# Layout Rules

- Single-page vertical rhythm with alternating wide sections and denser grids
- Hero uses asymmetric composition on desktop and stacked layout on mobile
- Content width capped for comfortable reading

# Component Patterns

- Section headers pair eyebrow labels with concise titles and muted supporting text
- Cards share the same panel treatment and hover lift behavior
- Timeline items use a vertical guide line with compact metadata blocks

# Interaction Rules

- Hover states raise contrast and slightly lift the component
- Focus states use visible accent outlines
- Motion remains subtle and duration stays short

# Accessibility Notes

- Maintain strong text contrast against dark surfaces
- Preserve readable font sizes on mobile
- Do not rely on color alone to communicate emphasis
